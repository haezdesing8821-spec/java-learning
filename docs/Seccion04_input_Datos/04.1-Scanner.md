# Tutorial De Java - Entrada De datos / Scanner Class

## 04.1 Entrada De Datos con `Scanner`en java

En Java, la `entrada de datos` desde el teclado se puede realizar utilizando la clase Scanner. Esta clase permite leer diferentes tipos de datos como cadenas, enteros, flotantes, etc.

```java
import java.util.Scanner;
public class EntradaDatos {
   public static void main(String[] args) {
       Scanner entrada = new Scanner(System.in);
       // Leer un String
       System.out.print("¿Cuál es tu nombre? ");
       String nombre = entrada.nextLine();
       System.out.println("Hola, " + nombre + ".");
       // Leer un entero
       System.out.print("Ingresa tu edad: ");
       int edad = entrada.nextInt();
       System.out.println("Tienes " + edad + " años.");
       // Leer un número flotante
       System.out.print("Ingresa tu altura (en metros): ");
       float altura = entrada.nextFloat();
       System.out.println("Tu altura es " + altura + " metros.");
       entrada.close();
   }
}
```

## Consideraciones importantes

Métodos comunes de Scanner: `nextLine()`: Lee una línea completa como cadena. `nextInt()`: Lee un número entero. `nextFloat()`: Lee un número flotante. `nextDouble()`: Lee un número decimal.

Manejo de excepciones: Si el usuario ingresa un tipo de dato incorrecto (por ejemplo, texto en lugar de un número), se generará una excepción. Es recomendable manejar estos casos con bloques `try-catch`.

Cierre del Scanner: Siempre cierra el objeto Scanner al finalizar su uso para liberar recursos.

Problema con `nextLine()` después de otros métodos: Si usas `nextLine()` después de métodos como `nextInt()`, puede capturar el salto de línea pendiente. Para evitarlo, utiliza una llamada adicional a `entrada.nextLine()` antes.

Este enfoque es ideal para programas interactivos que requieren entrada del usuario desde la consola.
