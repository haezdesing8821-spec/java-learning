## 04.2 Entrada De Datos con `JOptionPane` en java

* `JOptionPane` es parte del paquete javax.swing en Java y se utiliza para crear cuadros de diálogo estándar para la interacción con el usuario. Simplifica el proceso de mostrar mensajes, solicitar información o pedir confirmación a los usuarios. Los cuadros de diálogo creados por JOptionPane son modales, lo que significa que bloquean la interacción del usuario con otras ventanas hasta que el cuadro de diálogo se cierra.

## Características Clave

`JOptionPane` proporciona varios métodos estáticos para crear diferentes tipos de cuadros de diálogo:

`**showMessageDialog:**` Muestra un mensaje al usuario.

`**showConfirmDialog:**` Realiza una pregunta de confirmación (por ejemplo, Sí/No/Cancelar).

`**showInputDialog:*` Solicita al usuario que ingrese información.

`**showOptionDialog:**` Un cuadro de diálogo personalizable que combina las funcionalidades anteriores.

## Ejemplo de Uso

Aquí hay algunos ejemplos que demuestran el uso de `JOptionPane`:

### Mostrando un Cuadro de Mensaje (`showMessageDialog`):

```java
import javax.swing.JOptionPane;

public class Main {
   public static void main(String[] args) {
       JOptionPane.showMessageDialog(null, "This is an information message", "Information", JOptionPane.INFORMATION_MESSAGE);
   }
}
```

Esto muestra un cuadro de diálogo de información con un mensaje y un botón de `"Aceptar"`.

## Solicitando Confirmación

```java
int response = JOptionPane.showConfirmDialog(null, "Do you want to proceed?", "Confirm", JOptionPane.YES_NO_OPTION);
if (response == JOptionPane.YES_OPTION) {
   System.out.println("User chose YES");
} else {
   System.out.println("User chose NO");
}
```

Esto crea un diálogo con las opciones `"Sí"` y `"No"` y devuelve la elección del usuario.

## Solicitando entrada del usuario

```java
String input = JOptionPane.showInputDialog("Please enter your name:");
System.out.println("User entered: " + input);
```

Esto solicita al usuario que ingrese un valor y lo devuelve como una cadena.

## Diálogo de opción personalizada

```java
Object[] options = {"Option 1", "Option 2", "Cancel"};
int choice = JOptionPane.showOptionDialog(null, "Choose an option", "Custom Dialog",
       JOptionPane.DEFAULT_OPTION, JOptionPane.QUESTION_MESSAGE, null, options, options[0]);
System.out.println("User selected: " + choice);
```

Esto crea un diálogo con botones personalizados y devuelve el índice de la opción seleccionada.

## Parámetros y Personalización

`parentComponent:` El componente padre del cuadro de diálogo. Si es null, el cuadro de diálogo se centra en la pantalla.

`message:` El contenido que se mostrará en el cuadro de diálogo. Puede ser una cadena, un componente o un arreglo de objetos.

`messageType:` Define el tipo de mensaje (por ejemplo, `ERROR_MESSAGE`, `INFORMATION_MESSAGE`, `WARNING_MESSAGE`, `QUESTION_MESSAGE` o `PLAIN_MESSAGE`).

`optionType:` Especifica el conjunto de botones (por ejemplo, `YES_NO_OPTION`, `YES_NO_CANCEL_OPTION`, `OK_CANCEL_OPTION`).

`icon:` Un ícono personalizado para mostrar en el cuadro de diálogo.

`title:` El título de la ventana del cuadro de diálogo.

## Consideraciones

`S`eguridad de Hilos:` Los componentes de Swing, incluido JOptionPane, no son seguros para hilos. Utilice el Event Dispatch Thread (EDT) para actualizaciones.

`Comportamiento de Bloqueo:` Los cuadros de diálogo son modales y bloquean la interacción con otras ventanas hasta que se cierran.

`Componentes Personalizados:` Puede incrustar componentes personalizados como JTextField o JComboBox en el cuadro de diálogo para casos de uso avanzados.

`JOptionPane` es una herramienta versátil y fácil de usar para crear diálogos de usuario en aplicaciones Java.
