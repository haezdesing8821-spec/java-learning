## 07.7 Operadores De Precedencia en java

### ¿Qué es?

Cuando un cálculo contiene más de un operador, Java sigue reglas de orden de operaciones para decidir qué parte calcular primero.

Por ejemplo, la multiplicación ocurre antes de la suma:

```java
EjemploConsigue tu propio servidor Java
int result1 = 2 + 3 * 4;     // 2 + 12 = 14
int result2 = (2 + 3) * 4;   // 5 * 4 = 20

System.out.println(result1);
System.out.println(result2);
```

### ¿Por qué ocurre esto?

En , la multiplicación se realiza primero, así que la respuesta es .2 + 3 * 414

Si quieres que la suma ocurra primero, debes usar paréntesis: , que da .(2 + 3) * 420

Propina: Usa siempre paréntesis si quieres asegurarte de que el cálculo se hace en el orden que esperas. Además, facilita la lectura de tu código.`()`


### Orden de operaciones

Aquí tienes algunos operadores comunes, de mayor a menor prioridad:

`()`            - Paréntesis
`*, / , %`      - Multiplicación, división, módulo/%
`+, -`          - Suma, Resta-
`<>`, `>=`, `<=`- Comparación<>=<=
`==`, `!=`      - Igualdad!=
`&&`            - Y lógico
`||`            - O lógico
`=`             - Asignación
