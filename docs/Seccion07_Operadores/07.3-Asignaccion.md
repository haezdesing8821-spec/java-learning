# Tutorial De Java - Session 7. / Los Operadores

## 07.3 Operadores De Asignación en java

## ¿Qué es?

Los operadores de asignación se utilizan para asignar valores a variables.

En el ejemplo siguiente, usamos el operador de `asignación ()` para asignar el valor 10 a una variable llamada x:`=`

### Ejemplo

```java
int x = 10;
```

### Operadores de asignación

**NOTA IMPORTANTE:** Tipo de dato: int

En Java, cuando se usan enteros, la división (`/`) es división entera (se descarta el decimal).

| Operador | Ejemplo   | Equivalente               |
| -------- | --------- | ------------------------- |
| `=`      | `x = 5`   | Asigna el valor `5` a `x` |
| `+=`     | `x += 3`  | `x = x + 3`               |
| `-=`     | `x -= 2`  | `x = x - 2`               |
| `*=`     | `x *= 4`  | `x = x * 4`               |
| `/=`     | `x /= 2`  | `x = x / 2`               |
| `%=`     | `x %= 3`  | `x = x % 3`               |
| `&=`     | `x &= 3`  | `x = x & 3`               |
| `^=`     | `x ^= 3`  | `x = x ^ 3`               |
| `>>=`    | `x >>= 3` | `x = x >> 3`              |
| `<<=`    | `x <<= 3` | `x = x << 3`              |
| `|=`     | `x| = 3`  | `x = x | 3`               |

### EJEMPLO DE USO

```java
int x = 10;

x += 3; // x = 13
x -= 2; // x = 11
x *= 4; // x = 44
x /= 2; // x = 22
x %= 3; // x = 1
```

### OPERADOR DE bit a bitç

| Operador | Nombre        | Ejemplo      | Descripción                                                 |
| -------- | ------------- | ------------ | ----------------------------------------------------------- |
| `&`      | AND bit a bit | `a & b`      | Compara cada bit; devuelve `1` solo si **ambos bits son 1** |
| `|=`     | OR bit a bit| | `a |= b`      | b` | Devuelve `1` si **al menos uno** de los bits es 1      |
| `^`      | XOR bit a bit | `a ^ b`      | Devuelve `1` si los bits son **distintos**                  |
| `~`      | NOT bit a bit | `~a`         | **Invierte todos los bits** (complemento a uno)             |

### EJEMPLO DE OPERADOR bit a bit

```java
public class BitwiseExample {
    public static void main(String[] args) {

        int a = 10; // 00001010
        int b = 3;  // 00000011

        // AND bit a bit (&)
        int andResult = a & b;   // 00000010 -> 2

        // OR bit a bit (|)
        int orResult = a | b;    // 00001011 -> 11

        // XOR bit a bit (^)
        int xorResult = a ^ b;   // 00001001 -> 9

        // NOT bit a bit (complemento a uno) (~)
        int notResult = ~a;      // ~00001010 -> -11

        System.out.println("a & b  = " + andResult); // 2
        System.out.println("a | b  = " + orResult);  // 11
        System.out.println("a ^ b  = " + xorResult); // 9
        System.out.println("~a     = " + notResult); // -11
    }
}
```

### OPERADOR DE DESPLAZAMIENTO:

| Operación | Cálculo                           | Resultado |
| --------- | --------------------------------- | --------- |
| `x >>= 3` | `10 >> 3` → `00001010 → 00000001` | `1`       |
| `x <<= 3` | `10 << 3` → `00001010 → 01010000` | `80`      |

### EJEMPLO DE OPERADOR DE DESPLAZAMIENTO

```JAVA
int x = 10;

x &= 3;   // 2
x |= 3;   // 11
x ^= 3;   // 9
x >>= 3;  // 1
x <<= 3;  // 80
```

**Nota importante** (precisión técnica)

`>>` es desplazamiento aritmético (conserva el bit de signo)

`<<` desplaza a la izquierda y rellena con ceros

Existe también `>>>=` (desplazamiento lógico), no solicitado, pero definido en Java

**Nota:** La mayoría de los operadores de asignación son simplemente formas más cortas de escribir código. Por ejemplo, es igual que , pero más corto y a menudo más fácil de leer.`x += 5x = x + 5`
