## 07.6 Operadores Lógicos en java

### ¿Qué es?

Al igual que con los operadores de comparación, también puedes probar valores de or con operadores lógicos.`true` `false`

Se utilizan operadores lógicos para determinar la lógica entre variables o valores, combinando múltiples condiciones:`:`

**IMPORTANTE**

Tipo de datos: boolean

Valores:

    x = true

    y = false

Los operadores lógicos devuelven siempre un boolean

###  Operadores lógicos

| Operador | Descripción | Ejemplo (x = true, y = false) | Resultado |
| -------- | ----------- | ----------------------------- | --------- |
| `&&`     | AND lógico  | `x && y`                      | `false`   |
| `||`     | OR lógico   | `x || y`                      | `true`    |
| `!`      | NOT lógico  | `!x`                          | `false`   |

### EJEMPLO DE USO:

```JAVA
boolean x = true;
boolean y = false;

System.out.println(x && y); // false
System.out.println(x || y); // true
System.out.println(!x);     // false
```

### Nota técnica importante (precisión)

&& y || son operadores lógicos de cortocircuito (short-circuit):

En x && y, si x es false, y no se evalúa

En x || y, si x es true, y no se evalúa

Esto los diferencia de & y |, que no cortocircuitan cuando se usan con booleanos

### Ejemplo real: Comprobación de inicio de sesión

El siguiente ejemplo muestra cómo se pueden usar operadores lógicos en una situación real, por ejemplo, al comprobar el estado de inicio de sesión y los derechos de acceso:

```JAVA
EjemploConsigue tu propio servidor Java
boolean isLoggedIn = true;
boolean isAdmin = false;

System.out.println("Regular user: " + (isLoggedIn && !isAdmin));
System.out.println("Has access: " + (isLoggedIn || isAdmin));
System.out.println("Not logged in: " + (!isLoggedIn));
```
