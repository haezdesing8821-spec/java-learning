### 07.5 Operadores De Comparación en java

## ¿Qué es?

Los operadores de comparación se utilizan para comparar dos valores (o variables). Esto es importante en la programación, porque nos ayuda a encontrar respuestas y tomar decisiones.

El valor de retorno de una comparación es o bien . Estos valores se conocen como valores booleanos, y aprenderás más sobre ellos en los booleanos y si... Capítulo de Else.`true` `false`

En el siguiente ejemplo, usamos el operador mayor que `()` para determinar si 5 es mayor que 3:`>`

**IMPORTANTE**

Tipo de datos: int

Valores: a = 5, b = 8

Los operadores relacionales en Java siempre devuelven un boolean (true o false)

### Ejemplo

```java
int x = 5;
int y = 3;
System.out.println(x > y); // returns true, because 5 is higher than 3
```

### OPERADORES DE COMPARACIÓN

| Operador | Descripción   | Ejemplo (a = 5, b = 8) | Resultado |
| -------- | ------------- | ---------------------- | --------- |
| `==`     | Igual a       | `a == b`               | `false`   |
| `!=`     | Distinto de   | `a != b`               | `true`    |
| `>`      | Mayor que     | `a > b`                | `false`   |
| `<`      | Menor que     | `a < b`                | `true`    |
| `>=`     | Mayor o igual | `a >= b`               | `false`   |
| `<=`     | Menor o igual | `a <= b`               | `true`    |

### EJEMPLO DE USO

```java
int a = 5;
int b = 8;

System.out.println(a == b); // false
System.out.println(a != b); // true
System.out.println(a > b);  // false
System.out.println(a < b);  // true
System.out.println(a >= b); // false
System.out.println(a <= b); // true
```

**Nota técnica importante**

`==` compara valores en tipos primitivos

Para objetos (como String), == compara referencias, no contenido
(para contenido se usa equals())
