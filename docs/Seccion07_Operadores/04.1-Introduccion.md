# Tutorial De Java - Session 7. / Los Operadores

## 07.1 Introducción A LosOperadores en java

### ¿Qué son?

En Java, los `operadores` son símbolos especiales que indican al compilador que debe realizar operaciones específicas sobre uno o más operandos.

Estos permiten realizar diversas operaciones sobre datos y variables. Son esenciales para construir la lógica de los programas y abarcan desde cálculos matemáticos hasta evaluaciones lógicas y manipulaciones a nivel de bits. 

En el ejemplo siguiente, usamos el operador para sumar dos valores: `+`

```java
int x = 100 + 50;
```

Aunque el operador se utiliza a menudo para sumar dos valores, como en el ejemplo anterior, también puede usarse para sumar una variable y un valor, o una variable y otra variable:`+`

```java
int sum1 = 100 + 50;        // 150 (100 + 50)
int sum2 = sum1 + 250;      // 400 (150 + 250)
int sum3 = sum2 + sum2;     // 800 (400 + 400)
```

### Java divide los operadores en los siguientes grupos:

* Operadores aritméticos

* Operadores de asignación

* Operadores de comparación

* Operadores lógicos

* Operadores bit a bit
