## 07.2 Operadores Aridméticos en java

## ¿Qué es?

Los operadores en Java son símbolos especiales que permiten realizar diversas operaciones sobre datos y variables. Son esenciales para construir la lógica de los programas y abarcan desde cálculos matemáticos hasta evaluaciones lógicas y manipulaciones a nivel de bits.

Estos se dividn en varia categorías:

### 1. Opeaores Aridméticos

| Operador | Descripción     | Ejemplo (a=10, b=3) | Resultado |
| -------- | --------------- | ------------------- | --------- |
| `+`      | Suma            | `a + b`             | `13`      |
| `-`      | Resta           | `a - b`             | `7`       |
| `*`      | Multiplicación  | `a * b`             | `30`      |
| `/`      | División entera | `a / b`             | `3`       |
| `%`      | Módulo (resto)  | `a % b`             | `1`       |

### EJEMPLO

```JAVA
int x = 10;
int y = 3;

System.out.println(x + y); // 13
System.out.println(x - y); // 7
System.out.println(x * y); // 30
System.out.println(x / y); // 3
System.out.println(x % y); // 1
```

### Operadores de incremento y decremento en Java

**Importante:**

**Prefijo (`++a`, `--a`):** primero modifica la variable, luego la usa

**Sufijo (`a++, a--`):** primero usa la variable, luego la modifica

### 2. Incremento (`++`)
| Operador | Tipo    | Ejemplo (a=10) | Valor usado | Valor final de `a` |
| -------- | ------- | -------------- | ----------- | ------------------ |
| `++a`    | Prefijo | `++a`          | `11`        | `11`               |
| `a++`    | Sufijo  | `a++`          | `10`        | `11`               |

### EJEMPLO DE INCREMENTO

```java
int a = 10;

System.out.println(++a); // 11
System.out.println(a++); // 11 (luego a = 12)
```

### Decremento (`--`)

| Operador | Tipo    | Ejemplo (a=10) | Valor usado | Valor final de `a` |
| -------- | ------- | -------------- | ----------- | ------------------ |
| `--a`    | Prefijo | `--a`          | `9`         | `9`                |
| `a--`    | Sufijo  | `a--`          | `10`        | `9`                |

### EJEMPLO DE DECREMENTO

```java
int a = 10;
System.out.println(--a); // 11
System.out.println(a--); // 11 (luego a = 10)

```
