# Tutorial De Java: Introducción A Java Parte 2

## Introducción

En W3Schools, puedes probar Java sin instalar nada.

Su editor Java en línea se ejecuta directamente en su navegador y muestra tanto el código como el resultado:

**Código:**

```java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

**Resultado:**

```java
Hello World
```

Este editor se utilizará en todo el tutorial para demostrar los diferentes aspectos de Java.
Puedes escribir tu propio código Java en el editor y hacer clic en el botón "Run" para ver el resultado.
¡Diviértete programando en Java!

## Instalación de Java

Sin embargo, si desea ejecutar Java en su propia computadora, siga las instrucciones a continuación.

Es posible que algunas PC ya tengan Java instalado.

Para comprobar si tiene Java instalado en una PC con Windows, busque Java en la barra de inicio o escriba lo siguiente en el Símbolo del sistema (cmd.exe):

```bash
C:\Users\Your Name>java -version
```

Si Java está instalado, verá algo como esto:

```bash
java version "22.0.0" 2024-08-21 LTS
Java(TM) SE Runtime Environment 22.9 (build 22.0.0+13-LTS)
Java HotSpot(TM) 64-Bit Server VM 22.9 (build 22.0.0+13-LTS, mixed mode)
```

Si Java no está instalado, verá un mensaje de error que indica que el comando no se reconoce. Si no tiene Java instalado en su computadora, puede descargarlo en oracle.com .

**Nota:** En este tutorial, escribiremos código Java en un editor de texto. Sin embargo, es posible escribir Java en un entorno de desarrollo integrado (EDI), como IntelliJ IDEA, Netbeans o Eclipse, que resultan especialmente útiles al gestionar grandes colecciones de archivos Java.

Para instalar Java, siga estos pasos:

1. Vaya a la página de descarga de Java en [https://www.oracle.com/java/technologies/javase-jdk22-downloads.html](https://www.oracle.com/java/technologies/javase-jdk22-downloads.html).
2. Descargue el instalador adecuado para su sistema operativo (Windows, macOS, Linux).
3. Ejecute el instalador y siga las instrucciones en pantalla para completar la instalación.
4. Después de la instalación, es posible que deba configurar la variable de entorno `JAVA_HOME` y actualizar la variable `PATH` para incluir el directorio `bin` de Java.
5. Verifique la instalación abriendo una nueva ventana de Símbolo del sistema y escribiendo `java -version` nuevamente.
Ahora debería ver la versión de Java instalada en su sistema.
¡Felicidades! Ahora tienes Java instalado en tu computadora y estás listo para comenzar a programar en Java.

## Inicio rápido de Java

* En Java, cada aplicación comienza con un nombre de clase, y esa clase debe coincidir con el nombre del archivo.

* Vamos a crear nuestro primer archivo Java, llamado Main.java, lo cual se puede hacer en cualquier editor de texto (como el Bloc de notas).

El archivo debe contener un mensaje de "Hola mundo", que se escribe con el siguiente código:

principal.java

```java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

* Guarde el archivo como Main.java.
* Ahora, abra el Símbolo del sistema (cmd.exe) y navegue hasta el directorio donde guardó el archivo Main.java.
* Compile el archivo Java escribiendo el siguiente comando:

```bash
C:\Users\Your Name>javac Main.java
```

* Esto creará un archivo llamado Main.class en el mismo directorio.
* Ahora, ejecute el archivo compilado escribiendo el siguiente comando:

```bash
C:\Users\Your Name>java Main
```

* Debería ver el siguiente resultado:

```bash
Hello World
```

¡Felicidades! Acaba de crear y ejecutar su primer programa Java.
