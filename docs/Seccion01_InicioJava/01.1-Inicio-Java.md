# Tutorial de Java

## Aprende Java

Java es un lenguaje de programación popular. Java se utiliza para desarrollar aplicaciones móviles, aplicaciones web, aplicaciones de escritorio, juegos y mucho más.

### Ejemplo

```java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
    }
}
```

### Explicación del código

- `public class Main`: Declara una clase pública llamada `Main`.
- `public static void main(String[] args)`: Es el método principal de la aplicación, donde comienza la ejecución del programa.
- `System.out.println("Hello World");`: Imprime el texto "Hello World" en la consola.

Recomendamos leer este tutorial, en el orden que aparece en el menú de la izquierda.

Java es un lenguaje orientado a objetos y algunos conceptos pueden ser nuevos. Tómate descansos cuando sea necesario y repasa los ejemplos tantas veces como sea necesario.
