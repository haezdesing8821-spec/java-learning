# Tutorial De Java - Identificadores de Java

## Identificadores

Todas las variables de Java deben identificarse con nombres únicos.

Estos nombres únicos se llaman identificadores.

Los identificadores pueden ser nombres cortos (como x e y) o nombres más descriptivos (edad, suma, volumenTotal).

Nota: Se recomienda usar nombres descriptivos para Crear código comprensible y mantenible:

### Ejemplo 3

```java
// Good
int minutesPerHour = 60;

// OK, but not so easy to understand what m actually is
int m = 60;
```

### Las reglas generales para nombrar variables son

* Los nombres pueden contener letras, dígitos, guiones bajos y signos de dólar.
* Los nombres deben comenzar por letra.
* Los nombres deben empezar con una letra minúscula y no pueden contener espacios en blanco.
* Los nombres también pueden empezar por $ y _.
* Los nombres son sensibles a mayúsculas minúsculas ("myVar" y "myvar" son variables diferentes).
* Las palabras clave reservadas (como las palabras clave de Java, como o ) no pueden usarse como nombresintboolean.

## Identificadores inválidos

Aquí tienes algunos ejemplos de identificadores inválidos que podrían causar errores:

### Ejemplo 4

```java
// Invalid identifiers:
int 2ndNumber = 5;  // Cannot start with a digit
int my var = 10;    // Cannot contain spaces
int int = 20;       // Cannot use reserved keywords
```
