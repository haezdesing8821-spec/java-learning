# Tutorial De Java - Variables de impresión

## Variables de visualización

El método se utiliza a menudo para mostrar variables `.println()`

Para combinar texto y variable, utiliza el carácter: `+`

### Ejemplo

```java
String name = "John";
System.out.println("Hello " + name);
```

También puedes usar el carácter para añadir una variable a otra variable: `+`

### Ejemplo

```java
String firstName = "John ";
String lastName = "Doe";
String fullName = firstName + lastName;
System.out.println(fullName);
```

En Java, el símbolo tiene dos significados: `+`

* Para texto (cadenas), las une (llamada concatenación).

* Para los números, suma valores.

* Para valores numéricos, el carácter funciona como un operador matemático (fíjate que aquí usamos variables (enteras):+int

### Ejemplo

```java
int x = 5;
int y = 6;
System.out.println(x + y); // Print the value of x + y
```

A partir del ejemplo anterior, esto es lo que sucede paso a paso:

* x almacena el valor 5

* y almacena el valor 6

* `println()` muestra el resultado de , que es 11x + y

## Mezcla de texto y números

Ten cuidado al combinar texto y números en la misma línea de código. Sin paréntesis, Java tratará los números como texto después de la primera cadena:

### Ejemplo

```java
int x = 5;
int y = 6;

System.out.println("The sum is " + x + y);   // Prints: The sum is 56
System.out.println("The sum is " + (x + y)); // Prints: The sum is 11
```

## Explicación:

En la primera línea, Java combina con , creando la cadena . Entonces se suma a esa cadena, por lo que se convierte en .`"The sum is "x"The sum is 5"y"The sum is 56"`

En la segunda línea, los paréntesis se aseguran de que se calcule primero (resultando en ), Así que la salida es . `x + y11"The sum is 11"`
