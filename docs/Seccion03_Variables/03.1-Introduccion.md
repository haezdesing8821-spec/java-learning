# Tutorial De Java - Las Variables

## Variables en java

Las `variables` son contenedores para almacenar valores de datos.

En Java, existen diferentes tipos de variables, por ejemplo:

* **String** - almacena texto, como "Hola". Los valores de la cadena son rodeado de comillas dobles

* **int** - almacena enteros (números enteros), sin decimales, como 123 o -123

* **float** - almacena números de coma flotante, con decimales, como 19,99 o -19,99

* **char** - almacena caracteres individuales, como 'A' o 'B'. Los valores de char están rodeados de comillas simples

* **boolean** - almacena valores con dos estados: verdadero o falso

## Declaración (creación) de variables

Para crear una variable en Java, necesitas:

* Elige un tipo (como o intString)

* Dale un nombre a la variable (como , , o xagename)

* Opcionalmente, asignarle un valor usando `=`.

Aquí está la sintaxis básica:

### Sintaxis

```java
type variableName = value;
```

Por ejemplo, si quieres almacenar algo de texto, puedes usar un: `String`

### Ejemplo

Crea una `variable` llamada nombre de tipo y asignala el valor "John".

Luego usamos para imprimir la `variable` de nombre: `String println()`

```java
String name = "John";
System.out.println(name);
```

### Resultado

```java
John
```
Para crear una variable que deba almacenar un número, puedes usar: `int`

Ejemplo
Crea una variable llamada myNum de tipo y asigna el valor 15: `int`

```java
int myNum = 15;
System.out.println(myNum);
```

También puedes declarar una variable sin asignar el valor y asignarlo después:

### Ejemplo

```java
int myNum;
myNum = 15;
System.out.println(myNum);
```

### Resultado

```java
15
```

Ten en cuenta que si asignas un nuevo valor a una variable existente, sobrescribirá el valor anterior:

### Ejemplo

Cambia el valor de la `Variable: myNum` de 15 de a 20:

```java
int myNum = 15;
myNum = 20;  // myNum is now 20
System.out.println(myNum);
```

Variables finales
Si no quieres que otros (ni tú mismo) sobrescriban valores existentes, usa la palabra clave (esto declarará la variable como `"final"` o `"constante"`, lo que significa inmutable y de solo lectura): `final`

### Ejemplo

```java
final int myNum = 15;
myNum = 20;  // will generate an error: cannot assign a value to a final variable
```

## Otros tipos

Una demostración de cómo declarar variables de otros tipos:

### Ejemplo

```java
int myNum = 5;
float myFloatNum = 5.99f;
char myLetter = 'D';
boolean myBool = true;
String myText = "Hello";
```
