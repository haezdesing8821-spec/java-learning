# Tutorial De Java - Variables deJava - Ejemplos

## Ejemplos reales

A menudo, en nuestros ejemplos, simplificamos los nombres de las variables para que coincidan con su tipo de datos (`myInt` o `myNum` para tipos, `myChar` para tipos, y así sucesivamente). Esto se hace para evitar confusiones. `int char`

Sin embargo, como ejemplo práctico de uso de variables, hemos creado un programa que almacena diferentes datos sobre un estudiante universitario:

### Ejemplo 7

```java
// Student data
String studentName = "John Doe";
int studentID = 15;
int studentAge = 23;
float studentFee = 75.25f;
char studentGrade = 'B';

// Print variables
System.out.println("Student name: " + studentName);
System.out.println("Student id: " + studentID);
System.out.println("Student age: " + studentAge);
System.out.println("Student fee: " + studentFee);
System.out.println("Student grade: " + studentGrade);
```

## Calcular el área de un rectángulo

En este ejemplo real, creamos un programa para calcular el área de un rectángulo (multiplicando la longitud y el ancho):

### Ejemplo 8

```java
// Create integer variables
int length = 4;
int width = 6;
int area;

// Calculate the area of a rectangle
area = length * width;

// Print variables
System.out.println("Length is: " + length);
System.out.println("Width is: " + width);
System.out.println("Area of the rectangle is: " + area);
```
