# Tutorial De Java - Variables Multiples

## Declaración de Variables Multiples

Para declarar más de una variable del mismo tipo, puedes Utiliza una lista separada por comas:

### Ejemplo

En lugar de escribir:

```java
int x = 5;
int y = 6;
int z = 50;
System.out.println(x + y + z); // 61
```

Puedes escribir:

```java
int x = 5, y = 6, z = 50;
System.out.println(x + y + z); // 61
```

**Nota:** Declarar muchas variables en una línea es más corto, pero escribir una variable por línea a veces puede facilitar la lectura del código.

## Un valor para múltiples variables

También puedes asignar el mismo valor a varias variables en una sola línea:

### Ejemplo 2

```java
int x, y, z;
x = y = z = 50;
System.out.println(x + y + z); // 150
```
