# Tutorial De Java - Constantea (final)

## Constantes (palabra clave final)

Cuando no quieras que el valor de una variable cambie, usa la palabra clave.final

Una variable declarada con se convierte en constante, lo que significa que es inmutable y de solo lectura: `final`

### Ejemplo 5

```java
final int myNum = 15;
myNum = 20;  // Error: cannot assign a value to final variable 'myNum'
```

## ¿Cuándo usar el final?

Deberías declarar las variables cuando sus valores Nunca debería cambiar. Por ejemplo, el número de minutos en una hora o tu parto año:final

### Ejemplo 6

```java
final int MINUTES_PER_HOUR = 60;
final int BIRTHYEAR = 1980;
```

**Nota:** Por convención, las variables finales en Java suelen ser escrito en mayúsculas (por ejemplo, ). No es obligatorio, pero útil por la legibilidad del código y es común para muchos programadores.BIRTHYEAR
