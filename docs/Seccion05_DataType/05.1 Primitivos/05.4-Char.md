# Tutorial De Java - Session 5 / Tipos De datos

## 05.4 Tipos De Datos en java //Character

### Tipo Char

* El tipo de dato se utiliza para almacenar un solo carácter. El personaje debe ser rodeado de comillas simples, como `'A'` o `'c'`: `char`

### Ejemplo

```java
char myGrade = 'B';
System.out.println(myGrade);
```

* Alternativamente, si estás familiarizado con los valores `ASCII`, puedes usarlos para mostrar ciertos caracteres:

### Ejemplo

```java
char myVar1 = 65, myVar2 = 66, myVar3 = 67;
System.out.println(myVar1);
System.out.println(myVar2);
System.out.println(myVar3);
```

Nota: Se puede encontrar una lista de todos los valores ASCII en nuestra Tabla de Referencia ASCII.

### Cuerdas

* El tipo de dato se utiliza para almacenar una secuencia de caracteres (`texto`). Los valores de la cadena deben estar rodeados de comillas dobles: `String`

### Ejemplo

```java
String greeting = "Hello World";
System.out.println(greeting);
```

* El tipo de cadena se usa e integra tanto en Java que algunos lo llaman "el noveno tipo especial".

* Una cadena en Java es en realidad un tipo de dato no primitivo, porque se refiere a un objeto. El objeto `String` tiene métodos que se utilizan para realizar ciertos Operaciones sobre cuerdas. No te preocupes si aún no entiendes el término `"objeto"`. Aprenderemos más sobre cuerdas y objetos en un capítulo posterior.

### Ejemplo real

* Aquí tienes un ejemplo real de usar diferentes tipos de datos para calcular y generar el coste total de varios elementos:

### Ejemplo

```java
// Create variables of different data types
int items = 50;
float costPerItem = 9.99f;
float totalCost = items * costPerItem;
char currency = '$';

// Print variables
System.out.println("Number of items: " + items);
System.out.println("Cost per item: " + costPerItem + currency);
System.out.println("Total cost = " + totalCost + currency);
```
