## 05.3 Tipos De Datos en java // Booleanos

### Tipos booleanos

Muy a menudo, en programación, necesitarás un tipo de dato que solo pueda tener uno de dos valores, como:

    SÍ / NO
    ENCENDIDO / APAGADO
    VERDADERO / FALSO

Para esto, Java tiene un tipo de dato que solo puede tomar los valores o: `boolean` true or false

### Ejemplo

```java
boolean isJavaFun = true;
boolean isFishTasty = false;
System.out.println(isJavaFun);     // Outputs true
System.out.println(isFishTasty);   // Outputs false
```

* Los valores booleanos se utilizan principalmente para pruebas condicionales.
