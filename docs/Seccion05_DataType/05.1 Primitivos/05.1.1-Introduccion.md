# Tutorial De Java - Session 5 / Tipos De datos

## 05.1 Introducción a Los Tipos De Datos en java

* Los `Tipos De Datos` en `Java` son las categorías que definen qué tipo de `información` puede almacenar una `Variable`, como `números`, `texto` o `valores lógicos`. Éstos sirven para que el lenguaje sepa cuánta memória usar y qué operaciones están permitidas.

* En Java, los tipos de datos se dividen en primitivos y estructurados. Los primitivos son fundamentales y permiten manejar información básica como números, caracteres y valores booleanos. Los estructurados, por otro lado, son objetos que permiten trabajar con datos más complejos. 

## Tipos de Datos Primitivos

* **Java cuenta con ocho tipos primitivos:**

### Enteros: incluyen `byte`, `hort`, `int`, `long`, Estos se utilizan para manejar `números enteros`, tanto `positivos`como `negativos`. Por ejemplo:

```java
int numero = -100;
long distancia = 123456789L;
```

* **Flotantes:** Incluyen `float` y `double`, que permiten trabajar con números decimales. Ejemplo:

```java
float pi = 3.14f;
double e = 2.71828;
```

* **Caracteres:** El tipo char se utiliza para almacenar un `único carácter`. Ejemplo:

```java
char letra = 'A';
```

* **Booleanos:** El tipo boolean almacena valores lógicos (`true` o `false`). Ejemplo:

```java
boolean esJavaFacil = true;
```

* **String:** El tipo `String` son cadenas e texto almacena valores Alfabético, Alfanuméros, estos se definen entre comillas dobles (`" " `). Ejemplo:

```java
String eEjemplo = "Esto es una cadena o Stroing";
```

Los tipos estructurados son objetos que permiten almacenar múltiples valores o realizar operaciones avanzadas como `Clases`, `Arrays`, entre otros.

## No puedes cambiar el tipo

* Una vez que una variable se declara con un tipo, no puede cambiar a otro tipo más adelante en el programa:

### Ejemplo

```java
int myNum = 5;       // myNum is an int
// myNum = "Hello";  // Error: cannot assign a String to an int

String myText = "Hi"; // myText is a String
// myText = 123;      // Error: cannot assign a number to a String
```

**Nota:** Esta regla hace que Java sea más seguro, porque el compilador te detendrá si intentas mezclar los tipos por error.
