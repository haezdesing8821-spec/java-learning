# Tutorial De Java - Session 5 / Tipos De datos

## 05.2 Tipos De Datos en java // Números

### Números

* Los tipos de número primitivo se dividen en dos grupos:

Los tipos `enteros` almacenan números enteros, positivos o negativos (como `123` o `-456`), sin decimales. ¿Qué tipo deberías usar? depende del valor numérico. `byte`, `short`, `int` `long`.

Los tipos de coma `flotante` representan números con una parte fraccionaria, que contienen uno o más decimales. Existen dos tipos:`float` y `double`

Aunque hay muchos tipos numéricos en Java, los más usados son los números son (`para números enteros`) y (`para números de coma flotante`). Sin embargo, los describiremos a todos a medida que continúes.

## Tipos enteros

### Byte

* El tipo de dato puede almacenar números enteros de -128 a 127. Esto puede usarse en lugar de u otros tipos enteros para Guarda memoria cuando estés seguro de que el valor estará dentro de -128 y 127: `byte` `int`

### Ejemplo

```java
byte myNum = 100;
System.out.println(myNum);
```

### Corto

* El tipo de dato puede almacenar números enteros desde `-32768` hasta `32767`: `short`

### Ejemplo

```java
short myNum = 5000;
System.out.println(myNum);
```

### Int

* El tipo de dato puede almacenar números enteros desde `-2147483648` hasta `2147483647`. En general, y en nuestro tutorial, el tipo de dato es El tipo de dato preferido cuando creamos variables con un valor numérico. `int`

### Ejemplo

```java
int myNum = 100000;
System.out.println(myNum);
```

### Long

* El tipo de dato puede almacenar números enteros desde `-9223372036854775808` hasta `9223372036854775807`. Esto se usa cuando la inteligencia no es lo suficientemente grande para almacenar el valor. Ten en cuenta que debes terminar el valor con una `"L"`: `long`

### Ejemplo

```java
long myNum = 15000000000L;
System.out.println(myNum);
```

### Tipos  flotante

* Deberías usar un tipo de punto flotante siempre que necesites un número con decimal, como `9,99` o `3,14515`.

Los tipos de datos y pueden almacenar números fraccionarios. Ten en cuenta que debes terminar el valor con una `"f"` para los flotantes y una `"d"` para los dobles:

### Ejemplo de flotador

```java
* float myNum = 5.75f;
System.out.println(myNum);
```
### Doble ejemplo

```java
double myNum = 19.99d;
System.out.println(myNum);
```

¿Usar `float` o `double`?

* La precisión de un valor de coma flotante indica cuántos dígitos puede tener el valor después del punto decimal. La precisión de es solo `6-7` dígitos decimales, mientras que las variables tienen un precisión de unos 16 dígitos.`float` `double`

* Por lo tanto, es más seguro usarla para la mayoría de los cálculos. `double`

### Números científicos

* Un número de coma flotante también puede ser un número científico con una `"e"` que indica la `potencia de 10`:

### Ejemplo

```java
float f1 = 35e3f;
double d1 = 12E4d;
System.out.println(f1);
System.out.println(d1);
```
