## 05.5 Tipos De Datos en java // Keyword Var

### La palabra clave var

* La palabra clave se introdujo en Java 10 (`lanzado en 2018`). `var`

* La palabra clave permite al compilador detectar automáticamente el tipo de variable en función del valor que le asignes.`var`, Esto te ayuda a escribir código más limpio y evitar repetir tipos, especialmente para tipos largos o complejos.

Por ejemplo, en lugar de escribir , puedes escribir:`int x = 5`;

### Ejemplo

```java
var x = 5;  // x is an int
System.out.println(x);
```

Al usar , el compilador entiende eso es un .`var 5 int`

### Ejemplo con diferentes tipos

* Aquí tienes algunos ejemplos que muestran cómo se puede usar para crear variables de diferentes tipos, basadas en los valores que asignes:var

### Ejemplo

```java
var myNum = 5;         // int
var myDouble = 9.98;   // double
var myChar = 'D';      // char
var myBoolean = true;  // boolean
var myString = "Hello"; // String
```

### Notas importantes

* 1. Solo funciona cuando asignas un valor al mismo tiempo (no puedes declarar sin asignar un valor):`varvar x`;

```java
var x; // Error
var x = 5;  // OK
```

2. Una vez elegido el tipo, se mantiene igual. Ver ejemplo a continuación:

```java
var x = 5;  // x is now an int
x = 10;     // OK - still an int
x = 9.99;   // Error - can't assign a double to an int
```

### Cuándo usar var

* Para variables simples, suele ser más claro escribir el tipo directamente (, , , etc.).intdoublechar

* Pero para tipos más complejos, como o , puede hacer el código más corto y fácil de leer:`ArrayList`, `HashMapvar`
