## 05.2.1 Introducción a Los Tipos De Datos en java

## Tipos de Datos Estructurados

* En Java, los tipos de `datos estructurados` son aquellos que permiten almacenar y organizar múltiples valores (del mismo o de distinto tipo) bajo una sola entidad.
Se diferencian de los tipos primitivos (como `int, double, boolean`) porque pueden contener `colecciones de datos` o `objetos`.

### 1. Arreglos (Arrays)

    Estructuras de tamaño fijo que almacenan elementos del mismo tipo.
    Índices comienzan en 0.

### Ejemplo

```java
public class EjemploArray {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40};
        for (int num : numeros) {
            System.out.println(num);
        }
    }
}
```

### 2. Clases y Objetos

    Permiten agrupar datos (atributos) y comportamientos (métodos).
    Son la base de la Programación Orientada a Objetos en Java.

### Ejemplo:

```java
class Persona {
    String nombre;
    int edad;
}

public class EjemploClase {
    public static void main(String[] args) {
        Persona p = new Persona();
        p.nombre = "Ana";
        p.edad = 25;
        System.out.println(p.nombre + " tiene " + p.edad + " años.");
    }
}

```

## #3. Registros (Record)

    Introducidos en Java 14 (estables desde Java 16).
    Son clases inmutables y concisas para almacenar datos.

### Ejemplo:

```java
record Punto(int x, int y) {}

public class EjemploRecord {
    public static void main(String[] args) {
        Punto p = new Punto(5, 10);
        System.out.println("X: " + p.x() + ", Y: " + p.y());
    }
}

```

### 4. Colecciones (Collections Framework)

* Estructuras dinámicas que pueden crecer o reducirse.

* Incluyen:

    * List (ArrayList, LinkedList)
    * Set (HashSet, TreeSet)
    * Map (HashMap, TreeMap)

### Ejemplo:

```java
import java.util.ArrayList;

public class EjemploLista {
    public static void main(String[] args) {
        ArrayList<String> nombres = new ArrayList<>();
        nombres.add("Juan");
        nombres.add("María");
        nombres.add("Pedro");

        for (String nombre : nombres) {
            System.out.println(nombre);
        }
    }
}
```

### 5. Matrices Multidimensionales

Arreglos de arreglos.

### Ejemplo:

```java
public class EjemploMatriz {
    public static void main(String[] args) {
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6}
        };
        System.out.println(matriz[1][2]); // Imprime 6
    }
}

```

## ✅ Resumen:

* En Java, los tipos de datos estructurados más comunes son:

    * Arrays (unidimensionales y multidimensionales)

    * Clases y Objetos

    * Records

    * Colecciones (List, Set, Map)

    * Matrices (como arrays de arrays)
