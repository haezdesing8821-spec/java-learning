### 08.4 `Personajes Especiales`en java

### Cadenas - Caracteres especiales

Como las cadenas deben escribirse entre comillas, Java malinterpretará esta cadena, y generan un error:

```java
String txt = "We are the so-called "Vikings" from the north.";
```

La solución para evitar este problema es usar el **`carácter de escape con barra inversa`**.

El carácter de escape () de barra inversa convierte caracteres especiales en caracteres de cadena: `\`

| Escape character | Resultado | Descripción                   |
| ---------------- | --------- | ----------------------------- |
| `\'`             | `'`       | Comilla simple                |
| `\"`             | `"`       | Comilla doble                 |
| `\\`             | `\`       | Barra invertida (*backslash*) |

```java
char comillaSimple = '\'';
String texto = "Ella dijo: \"Hola\"";
String ruta = "C:\\Users\\Leo\\Documents";

System.out.println(comillaSimple); // '
System.out.println(texto);         // Ella dijo: "Hola"
System.out.println(ruta);          // C:\Users\Leo\Documents
```

**Nota técnica importante**

En Java:

* `'` se usa para char

* `"` se usa para String

Por eso deben escaparse cuando se quieren imprimir o almacenar literalmente

La secuencia inserta una comilla doble en una cadena: `\"`

### Ejemplo

```java
String txt = "We are the so-called \"Vikings\" from the north.";
```
La secuencia inserta una sola comilla en una cadena: `\'`

### Ejemplo

```java
String txt = "It\'s alright.";
```
La secuencia inserta una sola barra inversa en una cadena: `\\`

### Ejemplo

```java
String txt = "The character \\ is called backslash.";
```

### Otras secuencias de escape comunes válidas en Java son:

Características de escape de control en Java

| Code | Resultado       | Descripción                   |
| ---- | --------------- | ----------------------------- |
| `\n` | New Line        | Salto de línea                |
| `\t` | Tab             | Tabulación horizontal         |
| `\b` | Backspace       | Retroceso de un carácter      |
| `\r` | Carriage Return | Retorno al inicio de la línea |
| `\f` | Form Feed       | Salto de página               |
