## 08.1 `String`en java

### ¿Qué es?

Las cadenas se utilizan para almacenar texto.

Una variable contiene una colección de caracteres rodeados de comillas dobles ():String""

### Ejemplo

Crea una variable de tipo y asignala un valor:String

String greeting = "`Hello`";

### Longitud de cuerda

Una cadena en Java es en realidad un objeto, lo que significa que contiene métodos que pueden realizar ciertas operaciones sobre cadenas.

Por ejemplo, puedes encontrar la longitud de una cuerda con el método: `length()`

### Ejemplo

```java
String txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
System.out.println("The length of the txt string is: " + txt.length());
```

### Más métodos de cuerdas

Existen muchos métodos de cadenas disponibles en Java.

### Por ejemplo:

* El método convierte una cadena en letras mayúsculas.**`toUpperCase()`**

* El método convierte una cadena en letras minúsculas. **`toLowerCase()`**

#### Ejemplo

```java
String txt = "Hello World";
System.out.println(txt.toUpperCase());   // Outputs "HELLO WORLD"
System.out.println(txt.toLowerCase());   // Outputs "hello world"
```

### Encontrar un carácter en una cadena

El método devuelve el índice (la posición) de la primera aparición de un texto especificado en una cadena (incluyendo espacios en blanco): `indexOf()`

### Ejemplo

```java
String txt = "Please locate where 'locate' occurs!";
System.out.println(txt.indexOf("locate")); // Outputs 7
```

Java cuenta las posiciones desde cero. `0` es la primera posición en un cuerda, `1` es la segunda, `2` es la tercera ...

Puedes usar el método para acceder a un carácter en una posición específica de una cadena: `charAt()`

### Ejemplo

```java
String txt = "Hello";
System.out.println(txt.charAt(0));  // H
System.out.println(txt.charAt(4));  // o
```

### Comparando cuerdas

Para comparar dos cadenas, puedes usar el método: `equals()`

### Ejemplo

```java
String txt1 = "Hello";
String txt2 = "Hello";

String txt3 = "Greetings";
String txt4 = "Great things";

System.out.println(txt1.equals(txt2));  // true
System.out.println(txt3.equals(txt4));  // false
```

### Eliminación de espacios en blanco

El método elimina espacios en blanco desde el principio y el final de una cadena: `trim()`

### Ejemplo

```java
String txt = "   Hello World   ";
System.out.println("Before: [" + txt + "]");
System.out.println("After:  [" + txt.trim() + "]");
```

### Resultado

```java
Before: [   Hello World   ]
After: [Hello World]
```
