## 08.5 `Java Math`en java

### ¿Qué es?

La clase Java Math tiene muchos métodos que te permiten realizar tareas matemáticas sobre números.

### Math.max(x,y)

El método puede usarse para encontrar el valor más alto de x e y: `Math.max(x,y)`

### Ejemplo

```java
Math.max(5, 10);
```

### Math.min(x,y)

El método puede usarse para encontrar el valor más bajo de x e y: `Math.min(x,y)`

### Ejemplo

```java
Math.min(5, 10);
```

### Math.sqrt(x)

El método devuelve la raíz cuadrada de x: `Math.sqrt(x)`

### Ejemplo

```java
Math.sqrt(64);
```

### Math.abs(x)

El método devuelve el valor absoluto (positivo) de x: `Math.abs(x)`

### Ejemplo

```java
Math.abs(-4.7);
```

### Math.pow(x, y)

El método devuelve el valor de x elevado a la potencia de y: ```Math.pow(x, y)```

### Ejemplo

```java
Math.pow(2, 8);  // 256.0
```

**`Nota:`** significa multiplicado por sí mismo 8 veces:

```java
2 * 2 * 2 * 2 * 2 * 2 * 2 = 256Math.pow(2, 8)2
```

**`Nota:`** El método siempre devuelve un , incluso si el resultado es un número entero. Por ejemplo, devuelve (no )`.Math.pow()`, `doubleMath.pow(2, 8)256.0256`

### Métodos de redondeo

Java tiene varios métodos para redondear números:

**`Math.round(x)`** - redondea al entero más cercano

**`Math.ceil(x)`** - redondea hacia arriba (devuelve el número entero menor mayor o igual que x)

**`Math.floor(x)`*** - redondea hacia abajo (devuelve el mayor número entero menor o igual que x)

### Ejemplo

```java
Math.round(4.6);  // 5
Math.ceil(4.1);   // 5.0
Math.floor(4.9);  // 4.0
```

### Números aleatorios

`Math.random()` devuelve un número aleatorio entre `0.0 (inclusive)` y `1.0 (exclusivo)`:

### Ejemplo

```java
Math.random();
```

Para tener más control sobre el número aleatorio, por ejemplo, si solo quieres un aleatorio número entre 0 y 100, puedes usar la siguiente fórmula:

### Ejemplo

```java
int randomNum = (int)(Math.random() * 101);  // 0 to 100
```

**`Nota:`** devuelve un archivo . Para obtener un entero, necesitas lanzarlo con `.Math.random()`, `double(int)`.
