## 08.6 `Booleanos`en java

### Booleanos de Java

Muy a menudo, en programación, necesitarás un tipo de dato que solo pueda tener uno de dos valores, como:

    SÍ / NO

    ENCENDIDO / APAGADO

    VERDADERO / FALSO

Para esto, Java tiene un tipo de dato que puede almacenar valores.booleantruefalse

El nombre booleano proviene de George Boole, un matemático que definió por primera vez el sistema lógico utilizado en los ordenadores actuales.

### Valores booleanos

Un tipo booleano se declara con la palabra clave y solo puede tomar los valores o: `boolean` `true` O `false`

### Ejemplo

```java
boolean isJavaFun = true;
boolean isFishTasty = false;
System.out.println(isJavaFun);    // Outputs true
System.out.println(isFishTasty);  // Outputs false
```

En la práctica, los booleanos suelen ser el resultado de expresiones y se utilizan para probar condiciones en programas (véase más abajo).

### Expresiones booleanas

Una expresión booleana devuelve un valor booleano: `true` o `false`

Esto es útil para construir lógica y tomar decisiones en los programas.

Por ejemplo, puedes usar un operador de comparación, como el operador mayor que `()`, para averiguar si una expresión (o variable) es `verdadera` o `falsa`: `>`

### Ejemplo

```java
int x = 10;
int y = 9;
System.out.println(x > y); // returns true, because 10 is greater than 9
```

O incluso más fácil:

### Ejemplo

```java
System.out.println(10 > 9); // returns true, because 10 is greater than 9
```

En los ejemplos siguientes, usamos el operador igual a () para evaluar una expresión:==

### Ejemplo

```java
int x = 10;
System.out.println(x == 10); // returns true, because the value of x is equal to 10
```

### Ejemplo

```
System.out.println(10 == 15); // returns false, because 10 is not equal to 15
```

### Ejemplo de la vida real

Pensemos en un "ejemplo real" en el que necesitamos averiguar si una persona tiene edad suficiente para votar.

En el ejemplo siguiente, usamos el operador de comparación para determinar si la edad `()` es mayor que OR igual al límite de edad para votar, que se establece en : `25` `>=` `18`

### Ejemplo

```java
int myAge = 25;
int votingAge = 18;
System.out.println(myAge >= votingAge);
```

Un enfoque aún mejor sería envolver el código anterior en una sentencia, para poder realizar Acciones diferentes según el resultado: `if...else`

### Ejemplo

Salida "¡Suficientemente mayor para votar!" si es mayor o igual a . Si no, pon "No tengo edad suficiente para votar.": `myAge` `18`

```java
int myAge = 25;
int votingAge = 18;

if (myAge >= votingAge) {
  System.out.println("Old enough to vote!");
} else {
  System.out.println("Not old enough to vote.");
}
```

**Los booleanos son la base de todas las comparaciones y condiciones de Java.**
