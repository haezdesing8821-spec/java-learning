### 08.2 `Concatenar String`en java

### Concatenación de cadenas

El operador puede usarse entre cadenas a Combínalos. Esto se llama concatenación: `+`

### Ejemplo

```java
String firstName = "John";
String lastName = "Doe";
System.out.println(firstName + " " + lastName);
```
Ten en cuenta que hemos añadido un texto vacío (`" "`) para crear un espacio entre Nombre y Apellido en la impresión.

### Concatenación en oraciones

Puedes usar la concatenación de cadenas para construir oraciones con texto y variables:

### Ejemplo

```java
String name = "John";
int age = 25;
System.out.println("My name is " + name + " and I am " + age + " years old.");
```

### Resultado:

```java
My name is John and I am 25 years old.
```

### El método concat()

También puedes usar el método para concatenar cadenas: `concat()`

### Ejemplo

```java
String firstName = "John ";
String lastName = "Doe";
System.out.println(firstName.concat(lastName));
```

También puedes unir más de dos cadenas encadenando llamadas: `concat()`

```java
String a = "Java ";
String b = "is ";
String c = "fun!";
String result = a.concat(b).concat(c);
System.out.println(result);
```

**`Nota:`** Aunque puedes usar para unir varias cuerdas, La mayoría de los desarrolladores prefieren el operador porque es más corto y fácil de leer.`concat()`, `+`
