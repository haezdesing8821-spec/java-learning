### 08.3 `Números & String`en java

### Sumando números y cadenas

**¡ADVERTENCIA!**

Java utiliza el operador tanto para la suma como para la concatenación. `+`

Se suman números. Las cadenas se concatenan.

Si sumas dos números, el resultado será un número:

### Ejemplo

```java
int x = 10;
int y = 20;
int z = x + y;  // z will be 30 (an integer/number)
```

Si añades dos cadenas, el resultado será una concatenación de cadenas:

```java
String x = "10";
String y = "20";
String z = x + y;  // z will be 1020 (a String)
```

Si sumas un número y una cadena, el resultado será una concatenación de cadenas:

```java
String x = "10";
int y = 20;
String z = x + y;  // z will be 1020 (a String)
```
