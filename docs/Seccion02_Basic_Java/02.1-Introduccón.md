# Tutorial De Java - Sintaxis Básica

## Sintaxis de Java

* En el capítulo anterior, creamos un archivo Java llamado Main.javay usamos el siguiente código para imprimir "Hola mundo" en la pantalla:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hola mundo");
    }
}
```
* En este capítulo, explicaremos la sintaxis utilizada en el código anterior.

## Ejemplo explicado

Cada línea de código que se ejecuta en Java debe estar dentro de un class. El nombre de la clase siempre debe empezar con mayúscula inicial. En nuestro ejemplo, llamamos a la clase Main.

**Nota: Java distingue entre mayúsculas y minúsculas MyClassy myclassse trataría como dos nombres completamente diferentes.**

El nombre del archivo Java debe coincidir con el nombre de la clase. Por lo tanto, si se llama a la clase Main, el archivo debe guardarse como Main.java. Esto se debe a que Java usa el nombre de la clase para buscar y ejecutar el código. Si los nombres no coinciden, Java generará un error y el programa no se ejecutará.

Al guardar el archivo, guárdelo con el nombre .javade la clase y agréguelo al final. Para ejecutar el ejemplo anterior en su computadora, asegúrese de que Java esté correctamente instalado: Consulte el capítulo "Introducción" para saber cómo instalar Java. El resultado debería ser:

```Hola mundo
```
## El método principal

* El `main()` método es obligatorio en todo programa Java. Es donde el programa comienza a ejecutarse:

```java
public static void main(String[] args)
```

* **public:** El método main()debe ser público para que la máquina virtual de Java (JVM) pueda acceder a él desde fuera de la clase.

* **static:** El método `main()` debe ser estático para que la JVM pueda llamarlo sin crear una instancia de la clase.

* **void:** El método `main()`no devuelve ningún valor, por lo que se declara como void.

* **String[] args:** El método `main()` acepta un parámetro: una matriz de objetos String. Este parámetro se utiliza para pasar argumentos de línea de comandos al programa Java.

Cualquier código colocado dentro del main()método será ejecutado.

Por ahora, no necesitas comprender las palabras clave `public`, `staticy` o `void`profundamente. Las aprenderás más adelante en este tutorial. Solo recuerda: `main()` es el punto de partida de todo programa Java.

## Sistema.out.println()

* Dentro del método `main()`, podemos usar el `println()` método para imprimir una línea de texto en la pantalla:
**Ejemplo**

```java
public static void main(String[] args) {
  System.out.println("Hello World");
}
```

**Nota: Las llaves {}marcan el comienzo y el final de un bloque de código.**

`System.out.println()` Puede parecer largo, pero puedes pensarlo como un único comando u oración sque significa: "`Envía este texto a la pantalla`".

Esto es lo que significa cada parte (aprenderás los detalles más adelante):

`System` es una clase Java incorporada.
`out` es miembro de System, abreviatura de "salida".
`println()` es un método, abreviatura de "imprimir línea".
Por último, recuerda que cada declaración de Java debe terminar con un punto y coma `;`.
