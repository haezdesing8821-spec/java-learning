# Tutorial De Java - La Concatenación

## Concatenar en java

### ¿Qué es?

En `Java`, la `concatenación de cadenas` es el proceso de unir dos o más String para formar uno nuevo.
Existen varias formas de hacerlo, cada una con ventajas según el caso.

Aquí varioa ejemplos de `concatenación` en `java`:

## 1️⃣ Usando el operador + (forma más simple)

* **Qué es:** La forma más simple y conocida cde unir cadenas

* **Cómo funciona:** Si uno de los operandos es `String`, Java convierte el otro a cadena y los une. Se utiliza el moperador `+` para unir las variables del ejemplo acontinuación, se desea unir la variable `nombre` y `apellido` para logralo se introduce esto `+ " " +` en la declaración de una nueva variable cuyo valor o datoa almacenar sea la suma de ambas `Variables`  o dentro del metodo `println`:

```java
public class ConcatenacionEjemplo1 {
    public static void main(String[] args) {
        String nombre = "Juan";
        String apellido = "Pérez";
        // Concatenación con +
        String nombreCompleto = nombre + " " + apellido;    // las dobles comillas (" ") 
        // sirven para agregar el espacio que debe haber entre palabras

        System.out.println("Nombre : " +nombre);
        System.out.println("Apellido : " +apellido);
        System.out.println("Nombre completo: " +nombreCompleto);
    }
}
```
* **VENTAJAS:** Facil de leer.

* **DESVENTAJAS:** En bucles es ineficiente porque crea muchos objetos `String`.

## 2️⃣ Usando el método .concat()

* **¿Qué es:** Método propio de la clase `String`

* **Cómo funciona:** Llama directamente al metodo `concat()` de la clase `String`, que:
    - copia el contenido original (`variable saludo`).
    - copia el contenido del argumento (`variable nombre`).
    - devuelve una nueva cadena con ambas variables o cadenas unida.
    - no acepta `null`

**Ejemplo:**


```java
public class ConcatenacionEjemplo2 {
    public static void main(String[] args) {
        String saludo = "Hola";
        String nombre = "Marta";

        // concat() no modifica el String original, devuelve uno nuevo
        String mensaje = saludo.concat(" ").concat(nombre);

        System.out.println(mensaje);
    }
}
```
**VENTAJAS:** Más explicito que `+`

**DESVENTAJAS:** No acepta `null` y sigue siendo poco eficiente para muchas concatenaciones.

## 3️⃣ Usando StringBuilder (más eficiente en bucles)

* **Qué es:** Un objeto mutable diseñado para concatenar de forma eficiente

* **Cómo funciona:** Modifica un buffer mutable interno. No crea un `String` nuevo en cada operación; solo al final cuando llamas `toString()`

**Ejemplo:**

```java
public class ConcatenacionEjemplo3 {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();

        sb.append("Hola");
        sb.append(" ");
        sb.append("Mundo");

        String resultado = sb.toString();
        System.out.println(resultado);
    }
}
```
**VENTAJAS:** La opción más eficiente cuando concatenas dentro de `bucles`.

**DESVENTAJAS:** No es theard-safe.

## 4️⃣ Usando String.format() (para formato más controlado)

**Qué es:** Permite crear cadenas usando plantillas con formato.

**Cómo funciona:** Implementa una `plantilla` con marcadores (`%s`, `%d`, etc.) y reemplaza cadamarcador con los valores dado. Internamente usa `Formatter`, que es a su vez usa un `StringBuilder`.

```java
public class ConcatenacionEjemplo4 {
    public static void main(String[] args) {
        String nombre = "Ana";
        int edad = 25;

        String mensaje = String.format("Mi nombre es %s y tengo %d años", nombre, edad);
        System.out.println(mensaje);
    }
}
```
**VENTAJAS:** Ideal para aplicaciones.

**DESVENTAJAS:** Sintaxis menos intuitiva

## Recomendaciones

* Para pocas concatenaciones, usa `+` o `.concat()`.

* Para muchas concatenaciones en bucles, usa `StringBuilder` o `StringBuffer` (este último es seguro para hilos).

* Para mensajes con formato, usa `String.format()` o `System.out.printf()`.
