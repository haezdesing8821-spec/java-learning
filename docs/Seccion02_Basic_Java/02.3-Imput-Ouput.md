# Tutorial De Java - Entrada y Salida

## Texto impreso

Aprendió en el capítulo anterior que puede usar el método `println()` para generar valores o imprimir texto en Java:
**Ejemplo**

```java
System.out.println("Hola, Mundo!");
```

El código anterior imprime el texto `"Hola, Mundo!"` en la consola. El texto que desea imprimir debe estar entre comillas dobles `("")`.

Puedes agregar tantos métodos `println()` como quieras. Ten en cuenta que se agregará una nueva línea por cada método y cada uno debe terminar con un punto y coma `;`:
**Ejemplo**

```java
System.out.println("Hello World!");
System.out.println("I am learning Java.");
System.out.println("It is awesome!");
```

## Comillas dobles

El texto debe estar entre comillas dobles `""`.

Si olvida las comillas dobles, se produce un error:

Ejemplo

```java
System.out.println("This sentence will work!");
System.out.println(This sentence will produce an error);
```

El segundo `println()` método producirá un error porque el texto no está entre comillas dobles.

## El método Print()

También hay un método `print()` que es similar a `println()`.

**La única diferencia es que no inserta una nueva línea al final de la salida:**

### Ejemplo

```java
System.out.print("Hello World! ");
System.out.print("I will print on the same line.");
```

La salida del código anterior será:

```java
Hello World! I will print on the same line.
```

Tenga en cuenta que agregamos un espacio adicional (después de `"¡Hola mundo!"` en el ejemplo anterior) para una mejor legibilidad.

En este tutorial, solo lo usaremos `println()` porque hace que la salida del código sea más fácil de leer.
