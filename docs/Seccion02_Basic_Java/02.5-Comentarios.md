# Tutorial De Java - Comentarios

## Comentarios en java

Los `comentarios` se pueden usar para explicar el código Java y hacerlo más legible. También se pueden usar para evitar la ejecución al probar código alternativo.

**Comentarios de una sola línea**

Los comentarios de una sola línea comienzan con dos barras diagonales `(//)`.

Cualquier texto entre // y el final de la línea es ignorado por Java (no se ejecutará).

Este ejemplo utiliza un comentario de una sola línea antes de una línea de código:

### Ejemplo

```java
// This is a comment

System.out.println("Hello World");
```

Salida

```java
Hello World
```

Este ejemplo utiliza un comentario de una sola línea al final de una línea de código:

### Ejemplo

```java
System.out.println("Hello World"); // This is a comment
```

Salida

```java
Hello World
```

## Comentarios multilínea en Java

Los `comentarios` de varias líneas comienzan con `/*` y terminan con `*/`.

Cualquier texto entre `/*` y `*/` será ignorado por Java.

Este ejemplo utiliza un comentario de varias líneas (un bloque de comentarios) para explicar el código:

### Ejemplo:

```java
/* The code below will print the words Hello World
to the screen, and it is amazing */

System.out.println("Hello World");
```

Salida

```java
Hello World
```

## ¿Comentarios de una o varias líneas?

Tú decides cuál usar. Normalmente, lo usamos `//` para comentarios cortos y `/* */` para comentarios más largos.
