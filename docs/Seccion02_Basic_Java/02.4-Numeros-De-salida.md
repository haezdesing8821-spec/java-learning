# Tutorial De Java - Números de salida de Java

En Java, puede imprimir números de la misma manera que imprime texto, utilizando el `println()` método.

También puedes utilizar el método `println()` para imprimir números.

Sin embargo, a diferencia del texto, no colocamos números dentro de comillas dobles:
**Ejemplo**

```java
System.out.println(3);
System.out.println(358);
System.out.println(50000);
```

La salida del código anterior será:

```java
3
358
50000
```

Tenga en cuenta que no hay comillas dobles alrededor de los números en el código anterior.

También puedes realizar cálculos matemáticos dentro del método `println()`:

### Ejemplo

```java
System.out.println(3 + 3);
```

### Ejemplo:

```java
System.out.println(2 * 5);
```
