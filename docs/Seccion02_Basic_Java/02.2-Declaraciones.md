# Tutorial De Java - Sintaxis Básica **[Declaraciones]**

## Declaraciones

Un programa de computadora es una lista de "instrucciones" que deben ser "ejecutadas" por una computadora.

En un lenguaje de programación, estas instrucciones de programación se denominan declaraciones .

La siguiente declaración "instruye" al compilador a imprimir el texto "¡Java es divertido!" en la pantalla:

```java
System.out.println("Java es divertido!")
```

Cada instrucción de programación debe terminar con un punto y coma `;`, de lo contrario, el compilador generará un error.

### Ejemplo

```java
System.out.println("Java is fun!")
```

### Resultado

```java
error: ';' expected
```

En el ejemplo anterior, el compilador generó un error porque faltaba el punto y coma `;` al final de la instrucción.

**Consejo: Puedes pensar en una declaración como si fuera una oración en inglés. Al igual que las oraciones terminan con un punto ., las declaraciones en Java terminan con un punto y coma `;`.**

## Muchas declaraciones

La mayoría de los programas Java contienen muchas declaraciones.

Las sentencias se ejecutan, una por una, en el mismo orden en que están escritas:

### Ejemplo

```java
System.out.println("Hello World!");
System.out.println("Have a good day!");
System.out.println("Learning Java is fun!");
```

## Ejemplo explicado

Del ejemplo anterior, tenemos tres afirmaciones:

```java
System.out.println("Hello World!");
System.out.println("Have a good day!");
System.out.println("Learning Java is fun!");
```

1. La primera instrucción se ejecuta primero `(imprime "¡Hola mundo!" en la pantalla)`.
2. Luego, se ejecuta la segunda instrucción `(imprime "¡Que tengas un buen día!" en la pantalla)`.
3. Y, por último, se ejecuta la tercera instrucción `(imprime "¡Aprender Java es divertido!" en la pantalla)`.
