## 09.2 `Condicional if`en java

### 09.1.2 Condiciones de Java y sentencias If

Las condiciones y las sentencias if te permiten controlar el flujo de tu programa, decidiendo qué código se ejecuta y qué código se omite.

Piénsalo como en la vida real: si llueve, lleva un paraguas. Si no, no hagas nada.

Cada enunciado necesita una condición que resulte en o `.if` `true` `false`

Esto significa que las afirmaciones trabajan de la mano con los valores: `if` `boolean`

### Ejemplo

```java
boolean isRaining = true;

if (isRaining) {
  System.out.println("Bring an umbrella!");
}
```

### Con frecuencia, las condiciones se crean utilizando operadores de comparación, como los siguientes:

    Menos que: a < b

    Menor o igual a: a <= b

    Mayor que: a > b

    Mayor o igual a: a > = b

    Igual a: a == b

    No es igual a: a != b

Puedes usar estas condiciones para realizar diferentes acciones y tomar distintas decisiones.

Java tiene las siguientes sentencias condicionales:

Usa **`if`** para especificar un bloque de código a ejecutar, si una condición especificada es cierta 

Usa **`else`** para especificar un bloque de código a ejecutar, si la misma condición es falsa

Usa **`elif`**es para especificar una nueva condición a probar, si la primera condición es falsa

Usa **`switch`** para especificar muchos bloques alternativos de código a ejecutar

### La sentencia if

La instrucción `if` especifica un bloque de código a ejecutar si una condición es `true`

### Sintaxis

```java
if (condition) {
  // block of code to be executed if the condition is true
}
```

La condición dentro de la afirmación debe dar lugar a un valor: puede ser una expresión booleana o una variable booleana como: `if boolean x > yisLightOn`

También ten en cuenta que `If` está en minúsculas. Las letras mayúsculas `IF` generarán un error.

En el ejemplo siguiente, probamos dos valores para determinar si 20 es mayor que 18. Si la condición es `true`, imprime algún texto.

### Ejemplo

```java
if (20 > 18) {
  System.out.println("20 is greater than 18");
}
```

También puedes comparar variables:

### Ejemplo

```java
int x = 20;
int y = 18;
if (x > y) {
  System.out.println("x is greater than y");
}
```

### Ejemplo explicado

En el ejemplo anterior usamos dos variables, x e y, para comprobar si x es mayor que y (usando el operador). Como x es 20, y y es 18, y sabemos que 20 es mayor que 18, imprimimos en la pantalla que "x es mayor que y".>

La comparación también se utiliza a menudo para comprobar si dos valores son iguales, usando el operador `==`.

### Ejemplo

```java
int x = 20;
int y = 20;
if (x == y) {
  System.out.println("x is equal to y");
}
```

Aquí la condición es cierta, porque tanto como son 20, por lo que se imprime el mensaje "`x` es igual a `y`".`x == y`

### Uso de variables booleanas

También puedes probar variables booleanas directamente en una sentencia `if`.

### Ejemplo

```java
boolean isLightOn = true;

if (isLightOn) {
  System.out.println("The light is on.");
}
```

**`Nota:`** Escribir es lo mismo que escribir, pero más corto y fácil de leer.`if` (`isLightOn`), `if` (`isLightOn == true`)

Aquí tienes el mismo ejemplo con el valor `false` para ver que el programa continúa incluso cuando el bloque de código no se ejecuta.

### Ejemplo

```java
boolean isLightOn = false;

if (isLightOn) {
  System.out.println("The light is on.");  // This will not be printed
}

System.out.println("This line runs no matter what, because it is outside the if statement.");
```

Si no hay brackets
Si una sentencia `if` tiene solo una línea de código, Puedes escribirlo sin brackets `{ }` rizados.

### Ejemplo

```java
if (20 > 18)
  System.out.println("20 is greater than 18");
```

### Problema potencial

Sin brackets, solo la primera línea después de `if` la pertenece a él. Cualquier otra línea funcionará de todas formas, lo que puede dar lugar a resultados inesperados.

### Ejemplo

```java
int x = 20;
int y = 18;

if (x > y)
  System.out.println("x is greater than y");  // Belongs to if
  System.out.println("This line runs no matter what (not part of the if statement)");

// Output:
// x is greater than y
// This line runs no matter what (not part of the if statement)
```

### La Vía Segura

Para evitar errores, usa siempre brackets rizados. Esto deja claro qué líneas corresponden a la afirmación:{ }if

### Ejemplo

```java
int x = 20;
int y = 18;

if (x > y) {
  System.out.println("x is greater than y");
  System.out.println("Both lines are part of the if");
}

// Some code outside if
System.out.println("I am outside if, not part of if!");
```

**`NOTA:`** Usar siempre brackets `{ }` hace que tu código sea más claro, más fácil de leer y previene errores sutiles.
